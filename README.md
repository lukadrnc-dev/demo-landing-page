# Demo Landing Page Website

This repository contains a **2-page HTML/CSS demo website** designed as a portfolio-ready project for showcasing front-end web development skills. It includes a **Home (Landing) page** and a **Services page** with fully responsive design.

## Features

- Clean and modern design using HTML & CSS
- Responsive layout (mobile and desktop friendly)
- Navigation between Home and Services pages
- Contact section with placeholder email
- Simple, well-structured code for easy customization

## Pages

1. **index.html** – Landing / Home page  
2. **services.html** – Services page  
3. **style.css** – Shared CSS file for styling

## Live Demo

View the live demo here: [https://lukadrnc-dev.github.io/demo-landing-page/](https://yourusername.github.io/demo-landing-page/)

## How to Use

1. Clone or download the repository  
2. Open `index.html` in your browser  
3. Edit `style.css` or HTML files to customize content, colors, or layout

## Notes

- This project is intended as a **portfolio demo** for beginners in front-end development  
- No backend functionality is included  
- Safe to use as a reference for learning or small projects
